🎯 Morse Invaders

Mission: Shoot descending invaders by typing Morse code before they reach Earth!

Vibecoded from this original https://fritzsche.github.io/morse_invaders/

🎮 How to Play

• Only the red invader is active and sends Morse code

• Press ESC anytime to return to menu

⚙️ Game Modes

Letter Mode: Learn individual letters A-Z, 0-9

Word Mode: Practice with full words - each invader spells out one complete word in Morse code

You can start typing as soon as the word ends

📝 Custom Words
